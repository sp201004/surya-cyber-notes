### Linux Fundamentals
Linux is an open-source, highly reliable operating system that powers over 90% of the world's internet servers, cloud infrastructure, and the vast majority of cybersecurity hacking systems (such as Kali Linux).

Unlike Windows, Linux heavily emphasizes the Command Line Interface (CLI) for speed, automation, and remote administration.

#### Fundamental Commands to Navigate the Shell
* **`pwd` (Print Working Directory):** Shows your current directory location.
* **`ls` (List):** Lists the files and subdirectories inside your current folder. Use `ls -la` to see hidden files and details.
* **`cd` (Change Directory):** Navigates to a different folder (e.g., `cd /var/log`).
* **`cat` (Concatenate):** Reads and prints the contents of a text file to the terminal screen (e.g., `cat flag.txt`).
* **`whoami`:** Displays the username of the active terminal session.
